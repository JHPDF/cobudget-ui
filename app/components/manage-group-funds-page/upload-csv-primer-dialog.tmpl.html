Give many people funds at once using a CSV file

In order to give many people funds and / or invite them at the same time, you'll need to create a csv file to upload. How do I do that?

Please upload a csv file with 2 columns (no headings), with email addresses in the first column, and an amount that you want to ADD to their account in the second column. Check out the example below.

Example spreadsheet

PS. Don't worry, you'll be able to review and confirm changes before they are applied.
