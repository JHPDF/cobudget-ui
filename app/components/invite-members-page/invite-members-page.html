Manage Group Funds

Here you can change member's available funds using CSV files

Update by CSV file

Upload a csv file to update many member's funds at once. You can also invite new members at the same time. (more details)

Download CSV file

Download a csv of all your group members, their emails, and their available funds.

Using Safari?
A bug in Safari names all downloaded CSVs as Unknown. So after downloading, you will need to rename the file to YOUR_PREFERRED_NAME.csv.
